¿Qué es un componente en React y cuál es su principal ventaja?

Un componente en React es una función o clase de JavaScript que devuelve una parte de la interfaz de usuario (UI). Es como un bloque reutilizable que define cómo se debe ver y comportar una parte específica de la aplicación (por ejemplo, un botón, una tarjeta o un formulario). Ventaja principal: La reutilización del código. Puedes crear un componente una vez y usarlo muchas veces en diferentes partes de tu aplicación, lo que hace el código más ordenado, fácil de mantener y escalable.

¿Qué son las props y cómo se comunican los componentes a través de ellas?

Las props (propiedades) son parámetros que se envían de un componente padre a un componente hijo. Sirven para pasar información o datos y permiten que los componentes sean dinámicos y reutilizables.

¿Para qué se usa el método .map() en React y por qué necesitamos la prop key?

El método .map() se usa para recorrer arreglos (arrays) y renderizar listas de elementos en React. Por cada elemento del arreglo, .map() devuelve un nuevo elemento JSX que React muestra en pantalla.
